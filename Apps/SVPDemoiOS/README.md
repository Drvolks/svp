# SVPDemoiOS

Mini app iOS de test pour SVP:
- Champ URL (HTTP/HTTPS ou file://)
- Bouton `Load Video`
- Bouton `Play/Pause`
- Bouton `Start PiP` / `Stop PiP`

## Ouvrir

- Ouvre `Apps/SVPDemoiOS/SVPDemoiOS.xcodeproj` dans Xcode.
- Choisis un simulateur iOS 17+ ou un appareil.
- Build & Run.

## Notes

- PiP ne fonctionne pas sur tous les simulateurs.
- Si la lecture HTTP échoue, teste d'abord l'URL MP4 par défaut (fichier court).
